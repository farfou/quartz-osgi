# Quartz Osgi Example

An example of using Quartz in OSGI


## Getting started

### Setting Environment
 * cd com.wf.quartz.job.runtime
 * mvn install - This command copies project dependencies bundles direcotry under com.wf.quartz.job.runtime project.
 * import whole project to eclipse.
 * Have fun.
